## Graphql Nodejs Postgres PostMovies API

Simple GraphQL movie post API using Nodejs and Postgres

## Previous requirements

```
Nodejs
Docker
npm
```

## Project installation

```
git clone https://github.com/Googluu/Test1.git
cd Test1
npm i
```
