## Graphql Nodejs Postgres PostMovies API

Simple movie post API using GraphQL, Nodejs and Postgres

## Previous requirements

```
Nodejs
Docker
npm
```

## Project installation

```
git clone https://github.com/Googluu/Test1.git
cd Test1
npm i
```

## Add your origin

```
git remote -v

git remote add ...
```

```
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
```
